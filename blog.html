1. What is DOM and what is the purpose of DOM?

Ans: DOM means Document Object Model. We use this interface in any programming language for any web document. It represents the web page so that program can change its document style, content and structure by using HTML id or class name.

2. How will you select HTML elements using DOM? Name the DOM methods.

Ans: We can select any HTML element by using its specific id name. We can also select by class name but, this can't work for identify specifically. We can add in HTML elements any id name and then in JS we can call it by ".getElementById('idName')" There are much more DOM methods. such as: getElementId, getElementsByTagName,getElementByClassName,appendChild,removeChild,getAttribute,setAttribute,innerHTML,innerText,value etc.

3. What is event bubble?

Ans: Event bubble is a type of DOM event propagation. It is used for get any element which is inside the another element. Like parents and child node.We can use event bubble for calling parents node from the child node.

4. What is a callback function and why will you use it?

Ans: Callback function is a function which is to be executed after another function has finished execution.Any function that is passed an argument to another function so that it can be executed in another function as a callback function. We can use it for less time consuming and write short code and also use it lot of time in another function.
